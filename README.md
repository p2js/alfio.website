# alfio.website

This is my personal website, [alfio.website](https://alfio.website), built in Svelte.

Should you want to go digging around the source code for inspiration for your own website, some features of note are: 

- Good accessibility and adherence to web standards, with proper use of semantic elements, correct ogp `<meta>` tags and a "skip to content" link
- A mdsvex-based multi-route blog
- Components for a carbon.now.sh-inspired code snippet, proper figures and some blog layouts
